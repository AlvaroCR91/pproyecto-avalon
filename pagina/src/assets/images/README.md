# Carpeta de Imágenes

Esta carpeta está destinada a almacenar las imágenes que se utilizarán en el proyecto, especialmente para:

- Fondos de la página
- Iconos
- Logos
- Cualquier otro recurso visual

## Uso

Para usar una imagen como fondo, colócala aquí y luego referénciala en tus archivos CSS:

```css
body {
  background-image: url('/assets/images/nombre-de-tu-imagen.jpg');
}
```

## Formatos Recomendados

- **JPG/JPEG**: Para fotografías y fondos con muchos colores
- **PNG**: Para imágenes con transparencia o gráficos
- **SVG**: Para iconos y gráficos vectoriales
- **WebP**: Para mejor optimización (soporte moderno)

## Optimización

Recuerda optimizar tus imágenes antes de subirlas para mejorar el rendimiento:
- Comprimir imágenes grandes
- Usar resoluciones apropiadas para web
- Considerar formatos modernos como WebP
